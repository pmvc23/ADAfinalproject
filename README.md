# ADA Final Project – Labor Induction and Cesarean Delivery Analysis

This repository contains the project submission for Applied Data Analysis (ADA).  
The project evaluates whether induction of labor is associated with cesarean delivery among low-risk term pregnancies in rural Ghana.

## Contents
- `ADAprojectfinal.Rmd` — Full analysis code.
- `matdata.xlsx` — Original dataset
- `flowchart.png` — Sample size flow diagram .
- `README.md` — Overview of the project.

## Reproducibility
All analyses were performed using R  Version 2025.09.1+401 (2025.09.1+401) 
Required packages: tidyverse, readxl, broom, car, ResourceSelection, DiagrammeR.
Code provided in Rmd file

## Project Summary
The analysis found no significant association between induction and cesarean delivery.  
Previous cesarean section was the strongest predictor of cesarean delivery.
